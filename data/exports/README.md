# Exports Directory

This directory contains generated exports from the MadSpark Multi-Agent System.

## Export Formats

- **JSON**: Structured data exports
- **CSV**: Tabular data for analysis
- **Markdown**: Human-readable reports
- **PDF**: Formatted documents (when reportlab is available)

## File Naming Convention

All exports follow the pattern: `madspark_results_YYYYMMDD_HHMMSS.*`

## Note

Files in this directory are generated automatically and are excluded from version control.
